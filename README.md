# Sentic — Sentiment Analysis with DeepSeek

A clean, editorial‑style sentiment analysis dashboard.  
Paste reviews, feedback, tweets, or transcripts (one per line) and get:

- Per‑entry sentiment (+/neutral/-), score, confidence, emotions, key phrases  
- Overall metrics and distribution  
- Extracted themes  
- An executive summary, insights, and recommendations  

**Backend:** Python Flask + [DeepSeek API](https://platform.deepseek.com/)  
**Frontend:** Vanilla HTML/CSS/JS – no build step.
